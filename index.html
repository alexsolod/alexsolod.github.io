BFF's Lead Posting Instructions

Welcome to BFF's Lead Posting Instructions.



1) DESTINATION URL: The URL you will send leads to is: https://admin.bunnyfoofoo.app/api/v1/api_users/.

2) DATA TRANSMISSION METHOD: We accept HTTP POST method.

3) SUCCESS OR FAILURE: We provide a response of FAILURE. In cases of API key FAILURE we provide the "Bad credentials" error. In case of data failure, for example if you haven't send us required field, you will receive an empty array.

4) LEAD FIELDS: The following fields are the fields that we collect and store. Make specific note of the fields that are required since without that data, leads will not be received and will result in an instance of FAILURE. We do not collect or store any fields that are not in this table. Make sure you use the exact field names when passing data and be sure to urlencode your fields as well.


Example values:
{
"email": "string",
"first_name": "string",
"last_name": "string",
"website": "string",
"ip": "string",
"captured": "2018-08-16T10:47:38.524Z",
"lead_id": "string",
"zip": "string",
"state": "string",
"phone1": "string",
"job": "string",
"s1": "string",
"s2": "string",
"s3": "string",
"s4": "string",
"s5": "string"
}
Field Name Description Data Type Required
email The email address of the lead string YES
first_name The first name of the lead string YES
last_name The last name of the lead string YES
website The domain of the website the lead was generated from string NO
ip The IP Address of the lead when captured string NO
captured The last name of the lead string($date-time) NO
lead_id Your Universal Lead iD for the lead you are sending string NO
zip The zip code of the lead string NO
state The state the lead resides in string NO
phone1 The primary phone number of the lead (no dashes) string NO
job The job, industry or career search string of the lead string NO
